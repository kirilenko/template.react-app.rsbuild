# template.react-app.rsbuild

## Requirements

- [Node Version Manager](https://github.com/nvm-sh/nvm)
- [PNPM](https://pnpm.io/)

## Setup

Install the dependencies:

```bash
pnpm setup
```

## Get Started

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```
